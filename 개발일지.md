**추천 한잎 탭 이동 시 변경 버그** (`biteService.js`)

- 원인: 실제 경제일기는 `economic_journals` 테이블을 사용하는데 홈 체크 쿼리만 `diary`로 잘못 기재됨

**개발 서버 404 오류** (`vite.config.js`)
- `base: command === 'build' ? '/economing/' : '/'` → `base: '/economing/'` 고정
- 개발 서버(`localhost:5173`)에서 `/economing/` 경로로 접근 시 리소스를 찾지 못하던 문제 해결

- Supabase `user_quiz_results` 테이블 신규 생성 (RLS 포함):
