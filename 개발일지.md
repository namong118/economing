### 작업 내용

#### 버그 수정

**추천 한잎 탭 이동 시 변경 버그** (`biteService.js`)
- 홈 ↔ 경제한잎 탭 이동 시 `getRecommendedBite`가 매 마운트마다 Solar AI를 재호출해 추천 bite가 바뀌던 문제
- 모듈 레벨 `_recommendCache` 추가: `userId__userLevel__날짜` 키로 캐시 → 하루 한 번만 AI 호출, 이후 즉시 반환

**홈 화면 할일 체크 Supabase 404 오류** (`HomePage.jsx`)
- `Promise.all` → `Promise.allSettled` 교체: 하나의 쿼리가 실패해도 나머지 할일 체크가 독립적으로 동작
- `diary` → `economic_journals` 테이블명 수정 (잘못된 테이블 참조 버그)
- 원인: 실제 경제일기는 `economic_journals` 테이블을 사용하는데 홈 체크 쿼리만 `diary`로 잘못 기재됨

**개발 서버 404 오류** (`vite.config.js`)
- `base: command === 'build' ? '/economing/' : '/'` → `base: '/economing/'` 고정
- 개발 서버(`localhost:5173`)에서 `/economing/` 경로로 접근 시 리소스를 찾지 못하던 문제 해결

#### 퀴즈 결과 저장 기능 추가 (`EconomicBitePage.jsx`)
- 퀴즈 정답 시 `user_quiz_results` 테이블에 `{ user_id, bite_id, is_correct: true }` 저장
- `supabase` 클라이언트 import 추가
- Supabase `user_quiz_results` 테이블 신규 생성 (RLS 포함):
  ```
  id, user_id, bite_id, is_correct, created_at
  ```
- 홈 화면 "한잎 퀴즈 풀기" pill이 오늘 정답을 맞춘 경우 완료 체크로 표시됨

#### 온보딩 결과 화면 정리 (`OnboardingPage.jsx`)
- AI 결과 화면(`AIResultScreen`)에서 학습 로드맵 섹션 전체 제거
- "로드맵 자세히 보기 →" 버튼 제거
- 이모지 전체 제거 (헤더 🎉, 노밍 인트로 fallback 🌱, CompletionScreen 제목 🎉)
- 부제목 문구 변경: "당신만을 위한 경제 성장 로드맵이에요" → "이제 당신만의 경제 성장 여정을 시작해볼게요"

### Supabase 스키마 변경

- RLS 정책: 본인 데이터만 INSERT/SELECT 가능
